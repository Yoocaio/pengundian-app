# Aplikasi Pengundian Web Based - Jatis Mobile

## Project Structure
- `server/` - Backend API (Express.js + PostgreSQL)
- `cms/` - CMS Frontend (React + Vite)
- `landing/` - Landing Page Live Drawing (React + Vite)

## Tech Stack
- Backend: Node.js, Express, PostgreSQL, JWT, Multer, ExcelJS
- Frontend: React 18, Vite, React Router
- Deploy: Render (backend), Vercel (frontend)
